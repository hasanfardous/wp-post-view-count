== Description ==

The Post View Count plugin will record the number of views a post has received. It will display the view count for each post and shortcode with the copy facility by clicking in the admin post list table using custom columns. The shortcode that accepts a post ID and returns the view count to the bottom of post for the post with it's ID.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wp-post-view-count.zip` to the `/wp-content/plugins/` directory
2. Activate the **Post View Count** plugin through the **Plugins** menu in WordPress
